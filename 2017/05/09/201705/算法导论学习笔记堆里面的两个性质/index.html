每个孩子的子树的大小至多为2n/3

(最坏情况发生在树的最底层恰好为半满的时候)
规定总大小为n。当左子树为最大时，最底层恰好为半满。也就是最底层只有左子树有结点，右子树没有结点。
设底层半满时节点数为y，则再加y个节点，就是满树，并且根据满二叉树的特–若底层元素个数为x,则元素总数为2x-1.
所以n+y = 2y -1 +2y = 满树，所以n = 3y - 1
即左子树结点数= （满树-1）/2 = (4y-2)/2=2y-1= 2n / 3

对于任意包含n个元素的堆中，至多有[n/2^(k+1)]个高度为h的结点

最底层的元素个数为n/2个，这层高度为0；
倒数第二层元素个数为n/4个，高度为1；
倒数第三层元素个数为n/8个，高度为2；
如此类推，高度为h的，在倒数第h + 1层，元素个数为n / 2^(h + 1)。
