Java并发学习--信号量 (Semaphore)

Semaphore 是一个计数信号量。信号量维护了一个许可的数量。在请求许可可用前会阻塞每一个acquire()，获取许可后并使用完之后调用release()归还许可证。还可以用tryAcquire()方法尝试获取许可证。

在代码中，虽然有30个线程在执行，但是只允许10个并发的执行。
