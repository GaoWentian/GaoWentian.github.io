介绍

String类是final修饰，说明不可以被继承，String一旦被创建就不能改变，对String修改就会创建一个新的字符串，因此禁止使用循环修改字符串。

Sring在内存中的位置

使用new创建一个String对象，会在堆内分配内存，每个对象地址都不一样，而str3会先在字符串常量池查找，没有才会添加进去，如果存在直接引用地址。最后str6+str7是在JVM运行时才操作，会创建一个新的对象。
